Modeling Reputation Systems: Preliminaries

Louis Tiao

2015-05-09 00:26

Comments

Consider the following sets

\begin{equation*} U = \{\, u_i \mid i = 1, \dotsc, m \,\} \qquad \text{and} \qquad V = \{\, v_j \mid j = 1, \dotsc, n \,\} \end{equation*}

which may represent, respectively, a panel of judges and group of contestants, an array of sensors and set of physical measurements, a group of graders and set of exam papers, a number of customers and set of products, a group of film critics and a set of movies or broadly speaking, a set of \(m\) assessors and a set of \(n\) target items.

Consider the binary relation \(\mathcal{A} \subseteq U \times V\) over \(U\) and \(V\) where \((u, v) \in \mathcal{A}\) if and only if assessor \(u\) evaluated item \(v\). We denote the evaluation given by assessor \(u\) to item \(v\) by \(\chi(u, v)\) where \(\chi: \mathcal{A} \to \mathbb{R}\).

Unless \(\mathcal{A} = U \times V\), i.e. if \(\mathcal{A}\) is a proper subset of \(U \times V\), \(\chi\) is a partial function from \(U \times V\) to \(\mathbb{R}\) (\(\chi: U \times V \not \to \mathbb{R}\)) and is undefined for \((u, v) \notin \mathcal{A}\).

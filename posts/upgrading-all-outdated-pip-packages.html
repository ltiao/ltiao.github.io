Upgrading all outdated pip packages

Louis Tiao

2015-04-02 00:41

Comments

Source

Simply execute:

$ pip list --outdated | cut -d' ' -f1 | xargs pip install --upgrade

for any version of pip that has the list command (which I believe is 1.3+.)
